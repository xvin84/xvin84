## 🧰 Open-source утилиты

Три библиотеки под MIT, вынесенные из продакшена, - с тестами, CI и документацией:

- **[pgvector-rag](https://github.com/xvin84/pgvector-rag)** - async-тулкит RAG
  поверх PostgreSQL + pgvector: чанкинг по заголовкам, косинусный поиск с весами и
  порогом релевантности.
- **[tg-max-identity](https://github.com/xvin84/tg-max-identity)** - единая
  межплатформенная личность по хешу телефона с резолвером new/linked/already/conflict.
- **[retry-aiohttp-session](https://github.com/xvin84/retry-aiohttp-session)** -
  готовая aiogram-сессия с ретраями Bot API при сетевых сбоях.
